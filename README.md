## About The Project

gooberNet is a private social networking app designed for your personal group. It provides an Instagram-style feed where users can post content, scroll through updates, and stay connected within a closed community.

With its main selling point being the challenges section, where you can either create a custom challenge or participate in the active global one. The challenges as of current consist of: Color hunt

**Key Features:**

* Instagram-style social feed
* Post creation and media sharing
* Private group-based interaction
* Cloud-backed infrastructure for scalability
* In build challenges feature with easy posting and sharing of results

## Getting Started

### Prerequisites

* Flutter SDK

```bash
flutter doctor
```

### Installation

1. **Clone the repo**

```bash
git clone https://github.com/GalaxiMaster/gooberNet
```

2. **Install dependencies**

```bash
cd gooberNet
flutter pub get
```

3. **Run the app**

```bash
flutter run
```

## Usage

gooberNet allows members of your private group to share posts and interact through a centralised social feed. Users can create posts, browse updates, and stay connected in a streamlined, mobile-first interface.
